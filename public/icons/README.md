# Ícones do PWA

Para gerar os ícones PNG a partir do SVG:
1. Abra icon-512.svg no navegador
2. Faça screenshot e salve como icon-512.png (512x512)
3. Redimensione para icon-192.png (192x192)

Ou use: https://realfavicongenerator.net/

Substitua os arquivos icon-192.png e icon-512.png nesta pasta.
Para funcionar no PWA, os PNGs são obrigatórios.
